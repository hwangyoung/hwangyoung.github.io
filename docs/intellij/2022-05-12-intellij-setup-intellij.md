# IntelliJ 설정

```bash
# IntelliJ vim 설정
vi ~/.ideavimrc

set visualbell                # esc 누를 시 소리 안나게 하기
set noerrorbells              # esc 누를 시 소리 안나게 하기
set keep-english-in-normal    # insert 모드 => command 모드 전환 시 영어 유지
```
